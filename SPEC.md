# Especificações do Projeto — Introdução a Processamento de Imagens

Ferramenta para processar imagens nos moldes de um Photoshop simplificado.

## Linguagem e Convenções

- **Python**
- **Comentários em português**

## Requisitos Mínimos

1. Menu para carregar arquivo de imagem e salvar imagem modificada.
2. Algoritmos do curso:
   - Ampliação e redução de imagens
   - Transformações de intensidade
   - Filtragem espacial para suavização e aguçamento
   - Quanto mais recursos, melhor.
3. Interface gráfica (sugestões: TKinter, PyQt5).
